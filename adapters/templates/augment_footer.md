## 🆘 TROUBLESHOOTING

### Guidelines Not Loading
1. Check file location: `.augment-guidelines` or `.augment/rules/` directory
2. Verify file format: Markdown (.md extension)
3. Check character limit: Under 49,512 characters total
4. Restart Augment Code plugin if needed

### Guidelines Not Applying  
1. Verify rule type: "Always" rules apply automatically
2. Check plugin version: VSCode 0.492.0+ or JetBrains 0.197.0+
3. Ensure guidelines are in Agent/Chat mode (not Completions)
4. Clear Augment cache if persistent issues

### Performance Issues
1. Reduce guideline file size under character limits
2. Use specific, focused rules instead of generic advice
3. Split large guidelines into stack-specific files
4. Remove redundant or conflicting guidelines
