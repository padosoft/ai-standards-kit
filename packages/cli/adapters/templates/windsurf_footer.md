## 🆘 TROUBLESHOOTING

### Rules Not Loading
1. Check file location: `.windsurf/rules/` directory
2. Verify file format: Markdown (.md extension)
3. Check character limit: <12,000 characters per file
4. Restart Windsurf IDE if needed

### Rules Not Applying  
1. Verify activation mode: "Always On" + "Model Decision"
2. Check glob patterns: Ensure rules apply to correct file types
3. Review rule specificity: Too generic rules may be ignored
4. Clear Cascade cache if persistent issues

### Performance Issues
1. Split large rule files into focused, smaller files
2. Use specific glob patterns instead of wildcards
3. Remove redundant or conflicting rules
4. Optimize rule content for clarity and conciseness
